# 중소벤처기업진흥공단 정책자금 추천 서비스 — 기업 입력 항목 정의서

> **목적**: AI 에이전트가 기업 입력 데이터를 기반으로 Rule Engine 판별, 신청 적합도 스코어 산출, 정책자금 TOP5 추천, 리스크 분석을 수행하기 위한 입력 스키마 정의서입니다.
> **근거**: 중소벤처기업진흥공단 2026년 정책자금 융자계획 공고 및 융자대상·제한기업 기준 (https://www.kosmes.or.kr)

---

## 입력 항목 개요

| 섹션 | 항목 수 | 필수 여부 | 주요 용도 |
|------|--------|----------|----------|
| ① 기업 기본 정보 | 6개 | 필수 | 업종 분류, 성장단계 분류, 소상공인 판별 |
| ② 재무 정보 | 8개 | 필수 | 한계기업 판별, 부채비율 제한, 우대 조건 계산 |
| ③ 신용 및 세금 상태 | 4개 | 필수 | 즉시 제한 판별 |
| ④ 정책자금 수혜 이력 | 4개 | 필수 | 수혜 횟수 제한, 누적 한도 제한, 리스크 분석 |
| ⑤ 정책우선도 평가 지표 | 8개 | 선택 | 신청 적합도 스코어 핵심 요소 |
| ⑥ 자금 신청 희망 정보 | 2개 | 선택 | 자금 매칭 필터, 한도 안내 |
| **합계** | **32개** | 필수 22개 + 선택 10개 | |

---

## 용어 정의

- **`[BLOCK]`**: Rule Engine이 해당 자금 또는 전체 자금 신청을 즉시 제한하는 조건
- **`[SCORE]`**: AI 신청 적합도 스코어(0~100%) 산출에 반영되는 항목
- **`[FILTER]`**: 정책자금 TOP5 추천 시 자금 매칭 필터 조건으로 사용되는 항목

---

## ① 기업 기본 정보

> 기업의 업종, 규모, 위치 등 기본 식별 정보입니다. 성장단계 분류(창업기/성장기/재도약기) 및 소상공인 판별에 사용됩니다.

### 1-1. 업종 (KSIC 코드)

- **입력 타입**: 드롭다운 선택 또는 직접 입력 (한국표준산업분류 코드)
- **예시값**: `C2599` (기타 금속 가공제품 제조), `J6201` (컴퓨터 프로그래밍 서비스)
- **용도**: `[BLOCK]` `[SCORE]` `[FILTER]`
- **Rule 적용**:
  - 융자제외 대상 업종(별표1) 해당 시 → 전체 자금 신청 불가
  - 융자제외 업종 분류: 사행·향락업, 부동산 투기업, 공공운영업, 법무·세무·보건 등 전문서비스, 금융·보험업
  - 혁신성장분야(참고1), 초격차·신산업분야(참고1-1), 지역주력산업(참고2), 뿌리산업(참고3), 소재·부품·장비산업(참고4) 해당 시 → 중점지원분야로 우대 배정
  - 제조업 여부 → 소상공인 판별 기준 분기 (제조업: 10인 미만, 기타: 5인 미만)

---

### 1-2. 창업일

- **입력 타입**: 날짜 선택 (YYYY-MM-DD)
- **파생값**: 업력 (년·월 자동계산)
- **예시값**: `2019-03-15` → 업력 약 7년 2개월
- **용도**: `[SCORE]` `[FILTER]`
- **Rule 적용**:
  - 업력 7년 미만 → 부채비율 제한 예외 적용 가능
  - 업력 7년 미만 → 창업기반지원자금(혁신창업사업화) 매칭
  - 업력 7년 이상 → 신성장기반자금 매칭 우선
  - 업력 5년 초과 + 한계기업 조건 → 제한 강화
  - 업력 3년 미만 → 청년전용창업자금 대상 (대표자 연령과 함께 판별)
  - 업력 3년 미만 → 중진공 신용위험등급 CR1(최상위) 제한 예외 적용

---

### 1-3. 소재지 (시/도)

- **입력 타입**: 드롭다운 (17개 시·도)
- **예시값**: `경상남도`, `서울특별시`, `경기도`
- **용도**: `[SCORE]`
- **Rule 적용**:
  - 비수도권(서울·경기·인천 외) 소재 기업 → 전체 정책자금 예산의 60% 이상 비수도권 우선 배정으로 신청 적합도 가산
  - 지역별 정책금융기관협의회 추천 기업 → 일부 제한 요건 예외 적용 가능

---

### 1-4. 상시 근로자 수

- **입력 타입**: 숫자 입력 (명, 정수)
- **예시값**: `35`
- **용도**: `[BLOCK]` `[FILTER]`
- **Rule 적용**:
  - 소상공인 판별:
    - 광업·제조업·건설업·운수업: 10인 미만 → 소상공인 (대부분 제한)
    - 그 외 업종: 5인 미만 → 소상공인 (대부분 제한)
  - 소상공인 예외: 제조업, 혁신성장·초격차·신산업분야, 예비사회적기업, 협동조합, 마을기업, 소상공인 유예기업
  - 고용 성과 우대: 최근 1년간 10인 이상 신규 고용 → 누적 지원 한도 예외 적용 가능

---

### 1-5. 대표자 연령

- **입력 타입**: 숫자 입력 (만 나이, 정수)
- **예시값**: `38`
- **용도**: `[FILTER]`
- **Rule 적용**:
  - 만 39세 이하 + 업력 3년 미만 → 청년전용창업자금 신청 가능 (최대 1억원, 제조업 2억원, 연 2.5% 고정금리)

---

### 1-6. 휴·폐업 여부

- **입력 타입**: 라디오 (`운영 중` / `휴업 중` / `폐업`)
- **용도**: `[BLOCK]`
- **Rule 적용**:
  - 휴업 중 또는 폐업 → 전체 정책자금 신청 즉시 불가 (하드 블록)

---

## ② 재무 정보

> 최근 결산 기준 재무제표 데이터입니다. 한계기업 판별, 업종별 부채비율 제한, 우량기업 제한, 성과 우대 여부 계산에 사용됩니다.

### 2-1. 연간 매출액

- **입력 타입**: 숫자 입력 (원, 정수)
- **기준 연도**: 직전 결산 연도
- **예시값**: `2800000000` (28억 원)
- **용도**: `[SCORE]`
- **Rule 적용**:
  - 직전연도 매출 30억 이상 + 최근 3년 연평균 매출 증가율 20% 이상 → 매출 성과 우대 (누적 지원 한도 예외 적용 가능)
  - 수출기업화자금: 내수 매출 규모 기준 매칭 여부 판단

---

### 2-2. 자본총계

- **입력 타입**: 숫자 입력 (원, 정수)
- **예시값**: `1500000000` (15억 원)
- **용도**: `[BLOCK]`
- **Rule 적용**:
  - 자본총계 200억 원 초과 → 민간금융기관 이용 가능 우량기업으로 간주 → 제한
  - 예외: 소재·부품·장비 강소기업 100, 스타트업 100, 경쟁력강화위원회 추천 기업

---

### 2-3. 자산총계

- **입력 타입**: 숫자 입력 (원, 정수)
- **예시값**: `4000000000` (40억 원)
- **용도**: `[BLOCK]`
- **Rule 적용**:
  - 자산총계 700억 원 초과 → 민간금융기관 이용 가능 우량기업으로 간주 → 제한

---

### 2-4. 부채비율

- **입력 타입**: 숫자 입력 (%, 소수점 1자리)
- **계산식**: (부채총계 / 자본총계) × 100
- **예시값**: `180.5`
- **용도**: `[BLOCK]` `[SCORE]`
- **Rule 적용**:
  - 업종별 융자제한 부채비율(별표5) 초과 시 → 해당 자금 제한
  - 예외: 업력 7년 미만, 간편장부 사업자, 협동조합, 시설투자 기업(일정 조건), R&D 투자비율 1.5% 이상 기업

> **참고**: 업종별 기준 부채비율은 중진공 공고 별표5에 규정되어 있으며, 대체로 제조업 400~500%, 서비스업 200~300% 수준입니다. AI 에이전트는 KSIC 코드와 연동하여 해당 업종의 제한 부채비율을 자동 조회해야 합니다.

---

### 2-5. 영업이익

- **입력 타입**: 숫자 입력 (원, 음수 가능)
- **기준 연도**: 최근 3개 결산 연도 각각 입력 권장
- **예시값**: `250000000` (2.5억 원), `-50000000` (영업손실)
- **용도**: `[BLOCK]` `[SCORE]`
- **Rule 적용**:
  - 이자보상배율 = 영업이익 / 이자비용
  - 3년 연속 이자보상배율 1.0 미만 → 한계기업 → 업력 5년 초과 기업은 제한
  - 영업손실 발생 시 금융비용 없어도 이자보상배율 1.0 미만으로 간주

---

### 2-6. 이자비용

- **입력 타입**: 숫자 입력 (원, 0 이상)
- **예시값**: `80000000` (8천만 원)
- **용도**: `[BLOCK]`
- **Rule 적용**:
  - 영업이익과 함께 이자보상배율 계산에 사용
  - 이자비용이 0인 경우에도 영업손실 발생 시 이자보상배율 1.0 미만으로 처리

---

### 2-7. 자기자본 잠식 여부

- **입력 타입**: 라디오
  - `잠식 없음`
  - `부분 잠식` (자본총계 > 0이나 납입자본금보다 적음)
  - `완전 잠식` (자본총계 ≤ 0)
- **용도**: `[BLOCK]`
- **Rule 적용**:
  - 2년 연속 적자 기업 중 자기자본 완전 잠식 → 한계기업 → 업력 5년 초과 시 제한 (재창업자금은 신청 가능)

---

### 2-8. 직전 3개년 매출액 (선택)

- **입력 타입**: 숫자 입력 × 3 (3년 전, 2년 전, 1년 전 각 연도)
- **예시값**: `2000000000`, `2400000000`, `2800000000`
- **용도**: `[SCORE]`
- **Rule 적용**:
  - 연평균 매출 성장률 자동 계산
  - 최근 3년 연평균 매출 증가율 20% 이상 + 직전연도 매출 30억 이상 → 매출 성과 우대 조건 충족 여부 판별

---

## ③ 신용 및 세금 상태

> 즉시 제한 여부를 판단하는 하드 블록 조건들입니다. 이 섹션의 항목 중 하나라도 "있음"이면 해당 조건에 따라 전체 또는 일부 자금 신청이 즉시 불가합니다.

### 3-1. 국세·지방세 체납 여부

- **입력 타입**: 라디오 (`없음` / `있음`)
- **용도**: `[BLOCK]`
- **Rule 적용**:
  - 체납 중 → 전체 정책자금 신청 즉시 불가 (하드 블록)

---

### 3-2. 신용정보원 불량 정보 등록 여부

- **입력 타입**: 체크박스 다중 선택 (해당 항목 모두 선택)
  - `해당 없음`
  - `연체 정보 등록`
  - `대위변제·대지급 이력`
  - `부도 이력`
  - `관련인 등록`
  - `금융질서문란 등록`
  - `회생·파산 진행 중 또는 이력`
- **용도**: `[BLOCK]`
- **Rule 적용**:
  - 위 항목 중 1개라도 해당 → 전체 정책자금 신청 즉시 불가

---

### 3-3. 증권시장 상장 여부

- **입력 타입**: 라디오
  - `비상장`
  - `코스닥 상장`
  - `코스닥 기술특례상장 (상장 후 3년 이내)`
  - `유가증권시장(코스피) 상장`
- **용도**: `[BLOCK]`
- **Rule 적용**:
  - 코스닥 일반 상장 또는 코스피 상장 → 민간금융기관 이용 가능 우량기업 → 제한
  - 코스닥 기술특례상장 기업 상장 후 3년 이내 → 예외 (신청 가능)

---

### 3-4. 신용평가사 신용등급 (선택)

- **입력 타입**: 드롭다운 (`AAA` / `AA` / `A` / `BBB` / `BB` / `B` / `CCC 이하` / `미평가`)
- **용도**: `[BLOCK]`
- **Rule 적용**:
  - 신용평가사 BBB 이상 등급 → 민간금융기관 이용 가능 우량기업 → 제한
  - 미평가 또는 BB 이하 → 제한 없음 (다른 우량기업 기준 별도 적용)

---

## ④ 정책자금 수혜 이력

> 과거 정책자금 수혜 이력 데이터입니다. Rule Engine의 중복·횟수 제한 판별 및 리스크 분석(수혜 패턴 이탈 분석)에 사용됩니다.

### 4-1. 최근 5년간 중진공 정책자금 수혜 횟수

- **입력 타입**: 숫자 선택 (`0회` / `1회` / `2회` / `3회` / `4회` / `5회 이상`)
- **용도**: `[BLOCK]` `[SCORE]`
- **Rule 적용**:
  - 최근 5년 이내 3회 이상 → 추가 지원 원칙적 제한
  - 예외 (1회 추가 가능): 시설자금, 고용·수출·매출 성과 우대 기업, 소액 지원 기업(운전자금 5억 이하)
  - 예외 (2회 추가 가능): 중점지원분야 시설자금, 초격차 스타트업 프로젝트 선정 기업
  - 횟수 산정 제외 자금: 재도약, 긴급경영안정, 매출채권팩토링, 동반성장네트워크론, 이차보전, 신시장진출(수출국다변화), 마일스톤 등

---

### 4-2. 중진공 운전자금 누적 지원 금액

- **입력 타입**: 숫자 입력 (원, 0 이상)
- **예시값**: `1500000000` (15억 원)
- **용도**: `[BLOCK]`
- **Rule 적용**:
  - 운전자금 누적 25억 원 초과 → 운전자금 추가 지원 불가
  - 2018년 1월 2일 이후 신청 접수분에 한해 적용
  - 산정 제외: 재도약, 긴급경영안정, 스케일업금융, 매출채권팩토링, 동반성장네트워크론, 이차보전, 통상리스크대응긴급자금

---

### 4-3. 최근 5년간 정부·지자체 정책자금 총 수혜액

- **입력 타입**: 숫자 입력 (원, 0 이상)
- **참고**: 중소기업지원사업 통합관리시스템(sims.go.kr)에서 확인 가능
- **예시값**: `5000000000` (50억 원)
- **용도**: `[BLOCK]`
- **Rule 적용**:
  - 최근 5년간 합계 200억 원 초과 → 신규 지원 제한
  - 산정 제외: 재도약, 긴급경영안정, 매출채권팩토링, 동반성장네트워크론, 이차보전, 통상리스크대응긴급자금, 보증서부 융자 보증실적

---

### 4-4. 기업평가 탈락 또는 융자 결정 후 전액 포기 여부

- **입력 타입**: 라디오 + 시기 입력
  - `해당 없음`
  - `있음` → 날짜 입력 (YYYY-MM)
- **용도**: `[BLOCK]`
- **Rule 적용**:
  - 탈락 또는 전액 포기 후 6개월 미경과 → 해당 자금 재신청 제한
  - 예외: 신청 연도가 다른 경우, 실질기업주 변경 등 중대 변동 (1회 추가), 재도약·긴급경영안정 신청 전환, 이차보전 전환 후 은행 거절

## ⑤ 정책우선도 평가 지표

> 중진공 정책우선도 평가의 9개 지표에 대응하는 항목입니다. 이 섹션의 데이터가 AI 신청 적합도 스코어의 핵심 요소입니다. 모두 선택 입력이지만, 입력할수록 스코어 정확도가 향상됩니다.

### 5-1. 혁신성장 분야 해당 여부

- **입력 타입**: 체크박스 다중 선택
  - `AI·인공지능 관련`
  - `반도체·디스플레이`
  - `이차전지·전기차`
  - `바이오·헬스케어`
  - `로봇·자율주행`
  - `우주·항공`
  - `소재·부품·장비`
  - `뿌리산업·뿌리기술`
  - `Net-Zero·탄소중립 관련`
  - `AX(AI 전환) 도입·활용 기업`
  - `해당 없음`
- **용도**: `[SCORE]` `[FILTER]`
- **Rule 적용**:
  - 혁신성장분야 해당 → 연간예산 우선 배정, 신청 적합도 가산
  - AX 관련 기업 → 2026년 신설 AX 스프린트 우대트랙 (1,400억 규모) 매칭
  - Net-Zero 해당 → Net-Zero 유망기업 지원자금 매칭

---

### 5-2. 최근 1년 신규 고용 창출 인원

- **입력 타입**: 숫자 입력 (명, 정수)
- **기준**: 자금 신청 직전월 기준 12개월 전 대비 직전월 고용인원 증가분 (특수고용직 제외)
- **예시값**: `12`
- **용도**: `[SCORE]`
- **Rule 적용**:
  - 10인 이상 고용 창출 → 정책우선도 평가 가산
  - 10인 이상 고용 창출 → 운전자금 누적 한도(25억) 및 5년 3회 제한 예외 적용 가능

---

### 5-3. 최근 12개월 직·간접 수출실적

- **입력 타입**: 숫자 입력 (USD)
- **예시값**: `600000` (60만 달러)
- **용도**: `[SCORE]` `[FILTER]`
- **Rule 적용**:
  - 수출실적 50만 달러 이상 + 전년 동기 대비 20% 이상 증가 → 수출 성과 우대
  - 수출 성과 우대 → 누적 한도·횟수 제한 예외 적용 가능
  - 수출실적 보유 여부 → 신시장진출지원자금 매칭 조건

---

### 5-4. 보유 인증 현황

- **입력 타입**: 체크박스 다중 선택
  - `벤처기업 확인`
  - `이노비즈 (기술혁신형 중소기업)`
  - `메인비즈 (경영혁신형 중소기업)`
  - `소재·부품·장비 강소기업 100`
  - `스타트업 100`
  - `아기유니콘 또는 예비유니콘`
  - `초격차 스타트업 프로젝트 선정`
  - `도약(Jump-Up) 프로그램 선정`
  - `스마트공장 도입 확인`
  - `여성기업 확인`
  - `사회적기업 또는 예비사회적기업`
  - `해당 없음`
- **용도**: `[SCORE]` `[FILTER]`
- **Rule 적용**:
  - 소재·부품·장비 강소기업 100, 스타트업 100, 경쟁력강화위원회 추천 → 우량기업 제한 예외, 200억 초과 제한 예외
  - 아기유니콘·예비유니콘·초격차 스타트업 → 누적 지원 횟수·한도 제한 예외
  - 도약(Jump-Up) 프로그램 → 해당 자금 직접 매칭

---
